15.3 Miscellaneous settings

If you want a Jabber menu on the menu bar with some common commands, type M-x jabber-menu. You can remove it again with C-u M-x jabber-menu. Unfortunately, this cannot be changed through Customize settings, so you need to add (jabber-menu) to your .emacs to enable it permanently.
